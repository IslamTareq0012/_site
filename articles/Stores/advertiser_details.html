    Advertiser Details EndPoint

    For stores details retrieval, you can call api endpoint URL /api/v1/Store/Advertiser[?ID], using api keys obtained from the auth step.

    curl --location --request GET 'https://api.waffarx.com/api/v1/store/Advertiser?ID=3480' \
--header 'Authorization: __Bearer Token__' \
--header 'usertoken: __ User Token __ '

    Header Value
    Authentication Bearer Token
    UserToken [ User Token ]
